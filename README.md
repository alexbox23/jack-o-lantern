## basic-path-light.ino
Rainbow light with presence detection

### Components
* Adafruit Gemma
* NeoPixel ring
* PIR motion sensor
* 3.7V LiPo battery

## jack-o-lantern.ino
LED Candle Animation

### Components
* Arduino Uno
* NeoPixel ring
* PIR motion sensor
* 5V USB power bank

### Features
* Red flame pattern
* Sleep mode idle pattern
* Glows blue when presence is detected
