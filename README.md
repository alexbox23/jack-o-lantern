# Bare metal LEDs
LED projects powered by bare metal MCUs

## jack-o-lantern.ino
LED Candle Animation

### Components
* Arduino Uno
* NeoPixel ring
* PIR motion sensor
* 5V USB power bank

### Features
* Sleep mode
* Glows blue when presence is detected

## basic-path-light.ino
Rainbow light with presence detection

### Components
* Adafruit Gemma
* NeoPixel ring
* PIR motion sensor
* 3.7V LiPo battery
